No livro PHP: a Bíblia, Tim Cnverse explica o que é PHP.

O PHP é ima linguagem para a criação de scripts para a Web do lado servidor embutidos em HTML, cujo o código-font éaberto, e que é compatível com os mais importantes servidores Web (especialmente Apache). O PHP permite incorporar fragmentos de códigos em páginas HTML normais- codigos esse que é interpretado á medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem que "cola", facilitando a conexão de suas páginas Web com o banco de dados do lado do servidor.
